# 🛠️ Automação de Cadastro de Produtos

Este projeto demonstra como automatizar o login em uma página web e o cadastro de produtos a partir de um arquivo CSV utilizando **Python**.  
O objetivo é mostrar na prática como usar bibliotecas como `pyautogui` e `pandas` para automatizar tarefas repetitivas.

## ⚙️ Tecnologias utilizadas
- Python 3.x  
- [PyAutoGUI](https://pyautogui.readthedocs.io/en/latest/)  
- [Pandas](https://pandas.pydata.org/)  
- [OpenPyXL](https://openpyxl.readthedocs.io/en/stable/)  
- Módulos nativos: `os`, `time`


## 📂 Estrutura dos arquivos
- `aula 01 automação de tarefas.py` → Script principal de automação  
- `produtos.csv` → Base de dados dos produtos  
- `cadastro dos produtos.png` → Exemplo da tela de cadastro  
- `tabela com o cadastro dos produtos registrados.png` → Resultado após execução  

⚠️ Observações
Os valores de login e senha são apenas exemplos didáticos.

O script depende da posição dos elementos na tela (coordenadas x, y), então pode ser necessário ajustar conforme sua resolução.

O projeto é para fins de estudo de automação com Python.
